[Hoe installeer ik de clanpack]
Stap 1: Open je GTA installatie folder 
Stap 2: Open de map GTA V in de clanpack
Stap 3: Sleep de x64 map in jouw GTA map (Hij vraag waarschijnlijk om iets te vervangen, doe dit)
Stap 4: Open de map waarin FiveM is geinstalleerd (Rechter muisknop op je FiveM icoon, druk op bestandslocatie openen)
Stap 5: Sleep de FiveM Application Data in dezelfde map als waar je net bent ingekomen (Als hij vraag om iets te vervangen, doe dit dan)
